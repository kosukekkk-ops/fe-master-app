## 説明（Description・4000字以内）

```
基本情報技術者試験（FE）の対策アプリです。「問題を解く → 間違えたところの用語が自動で苦手リストにたまる → 単語帳で復習する」という流れで、弱点をつぶしながら合格力を高めます。

■ 特長

・大量の演習問題（科目A・科目B 合わせて3,000問以上）
　用語の定義問題、計算問題（稼働率・基数変換・損益分岐点など）、科目Bのアルゴリズム（擬似言語トレース）まで幅広く収録。

・誤答から苦手単語を自動蓄積（このアプリの中心機能）
　間違えると、その問題に関わる重要用語が「苦手単語帳」へ自動で登録され、間違えた回数も記録されます。正解した問題でも、あやふやなら手動で登録できます。

・丁寧な解説
　各用語は「なぜそうなるのか」「実務ではどう使われるか」まで踏み込んで解説。アルファベットの略語は正式名称（何の略か）も表示します。

・単語帳（フラッシュカード）
　全用語／苦手用語をタップでめくって復習。シャッフル表示や、覚えた単語を非表示にする機能つき。

・学習ログ
　9分野（基礎理論・アルゴリズム・ハードウェア・データベース・ネットワーク・セキュリティ・開発管理・経営戦略・会計法務）別の正答率をレーダーチャートで可視化。正答率の推移や、間違えた回数トップ10も確認できます。

・オフライン対応
　問題演習・単語帳・学習ログはすべて通信不要。通学・通勤中でも使えます。

・広告なし・アカウント登録なし
　学習データは端末内にのみ保存され、外部に送信されません。

■ こんな方へ
・基本情報技術者試験の合格を目指している方
・スキマ時間に用語や計算問題を反復したい方
・自分の弱点分野を「見える化」して重点的に対策したい方

※ 本アプリは試験対策の学習教材です。試験の実施団体とは関係ありません。
```

## プライバシー（App Privacy）

このアプリは**データを一切収集しません**（学習データは端末の localStorage に保存されるのみ、サーバー送信なし、解析SDKなし、広告なし）。

App Store Connect のプライバシー設定では次を選択：

- **Data Collection: "No, we do not collect data from this app"**（データを収集しない）

これで「データは収集されません（Data Not Collected）」のプライバシーラベルになります。

## スクリーンショット（別途、実機またはシミュレータで撮影）

App Store は最低 **6.9型 iPhone（1320×2868 など）** のスクショが必要です。以下5枚を推奨（アプリの主要画面）：

1. ホーム画面（学習状況ダッシュボード）
2. 演習中の4択問題画面
3. 誤答時の解説＋苦手単語登録の画面
4. 単語帳（フラッシュカード）画面
5. 学習ログ（9分野レーダーチャート）画面

各スクショに一言キャッチ（例：「間違えるほど、苦手が見える。」）を上部に載せると訴求力が上がります。撮影は Codemagic ビルド後の TestFlight 実機、または後日ローカルで用意します。
